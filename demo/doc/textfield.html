TextField - 文本输入框

牛佳庆

<njq.niu@hand-china.com>

文本输入框.输入框有只读,必输等状态.

输入框可以绑定到某个DataSet上,通过设置DataSet的Field的一些属性来设置对应的输入框的状态.

例 1. 代码示例

<textField id="text4" required="true" prompt="输入框" emptyText="必须输入" width="150" />

表 1. 属性说明
属性名 说明
id 组件ID
bindTarget 绑定的DataSet的id
name 绑定到DataSet的Field的name
width 宽度
prompt 输入框的标签
emptyText 提示信息
required 是否必输
readonly 是否只读
